What is Routing?

  • URL is mapped to a view
  • View is a cached HTML template
  • View is managed by a controller
AngularJS comes with the ngRoute module.

ngRoute

Inject module and configure routes:

angular.module('app', ['ngRoute'])
	.config(['$routeProvider',
	function($routeProvider) {
		$routeProvider.when('/view1', {
			templateUrl: 'view1.html',
			controller: 'View1Controller'
		});
	}]);

ngRoute

Add directive to HTML page:

<div ng-view></div>

ngRoute

Demo

ngRoute

  • $routeProvider - map URL to template/controller
  • $routeParams - combination of $location's search() and path(), with path params taking precedence over search params when name collisions occur
  • $routeChangeSuccess - event available to $rootScope
  • $templateCache - where templates are cached

ngRoute

Limitations:
  • No parallel/sibling views - only one ng-view on an HTML page
  • No named views
  • No nested views
  • Can't switch to a view without changing the URL
Angular-UI provides the UI-Router module.

UI-Router

https://github.com/angular-ui/ui-router

  • App becomes a state machine
  • Organize the app around states
  • Each state may have an optional URL route
  • States are bound to named, sibling/parallel, and nested views

UI-Router

Inject module and configure states:

angular.module('app', ['ui.router'])
	.config(['$stateProvider',
	function($stateProvider) {
		$stateProvider.state('state1', {
			url: '/view1',
			templateUrl: 'view1.html',
			controller: 'View1Controller'
		});
	}]);

UI-Router

Add directive to HTML page:

<div ui-view="home"></div>

UI-Router

Demo

UI-Router

  • $stateProvider - map state to views/controllers and optional URL
  • $urlRouterProvider - define rules for matching a url to a state
  • $state - current state, go/transition to a state, determine if equal to or child of a state
  • $stateParams - url params and custom params available to the current state.
  • $stateChangeSuccess - event available to $rootScope
  • ui-sref directive - use in place of href to map anchor tag click to a state
  • $templateCache - where templates are cached

grunt-angular-templates

https://www.npmjs.org/package/grunt-angular-templates

  • Grunt build plugin
  • Speed up your AngularJS app by automatically minifying, combining, and caching your HTML templates with $templateCache.
  • Template HTML is embedded in JavaScript.
  • Eliminate the need to download each template HTML file separately.
  • Works well with grunt-usemin.

grunt-angular-templates

Code generated and appended to your combined JavaScript before minification:

angular.module('app').run(["$templateCache", function($templateCache) {
	$templateCache.put("home.html",
		// contents for home.html ...
	);
	...
	$templateCache.put("preferences.html",
		// contents for preferences.html
	);
}]);

UI-Router Extras

http://christopherthielen.github.io/ui-router-extras

  • Sticky/Parallel State
  • Deep State Redirect
  • Future State
  • Previous State

UI-Router Extras

Inject module and configure states:

angular.module('app', ['ui.router', ct.ui.router.extras'])
.config(['$stateProvider', '$urlRouterProvider',
function($stateProvider, $urlRouterProvider) {
	$stateProvider.state('foo.bar', {
		views: { ... },
		deepStateRedirect: true,
		sticky: true
	});
}]);
